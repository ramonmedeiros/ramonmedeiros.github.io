# ramonmedeiros.github.io

Personal blog at [ramonmedeiros.dev](https://ramonmedeiros.dev). Built with [Astro](https://astro.build).

## Development

```bash
make run      # start dev server at localhost:4321
make build    # production build → dist/
make preview  # preview production build locally
```

## Deploy

Pushes to `master` automatically deploy via GitHub Actions → GitHub Pages.

> **Note:** GitHub Pages must be set to **Source: GitHub Actions** in repo Settings → Pages.
